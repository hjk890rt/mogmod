HOW TO BUILD:

1. Install JDK 17
2. Download Fabric example mod (or use Gradle Fabric template)
3. Replace src files with this project
4. Run: ./gradlew build
5. Find jar in build/libs

WHAT IT DOES:
- Press G (changeable in controls)
- Opens creative inventory ONLY
- No flight, no insta-break, no health changes
- Locked to your UUID

IMPORTANT:
- Items can still be taken from creative menu (intended)
- World remains survival (advancements OK)
